Design web pages with CSS

What is CSS?

CSS stands for Cascading Style Sheets, since HTML was just the structure of a web page CSS is putting all of the details on the structure.

How does it work?

The thing to pay attention too trying to understand how CSS works is there is a box around all of the HTML elements.
CSS works by cooperating with the rules of HTML elements. These rules control how the content of specified elements will be shown throughout the site. The rules of CSS contains two parts. One being a Selector and the other being a declaration.
